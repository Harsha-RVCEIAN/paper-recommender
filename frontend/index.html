Project Overview

Lightweight search interface over a curated research-paper dataset. Uses client-side trie for instant autocomplete and a Flask backend for search and data delivery.

Dataset

Contains paper metadata (title, authors, year, keywords, abstract, references). Citation counts shown are computed by the backend from references.

Features

  • Keyword-only search (AND semantics)
  • Autocomplete with Tab-accept (like Google)
  • Citation-aware suggestions
