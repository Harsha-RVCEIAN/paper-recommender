# 📄 Paper Recommender System

A citation-aware research paper recommendation system that helps users discover relevant academic papers based on keywords and contextual relevance.

---

## 🚀 Overview

This project is built to solve a simple but real problem:

> Finding relevant research papers is noisy, time-consuming, and inefficient.

Instead of blindly searching, this system:

* Uses keyword-based search
* Enhances results with citation awareness
* Provides a structured way to explore research papers

---

## 🧠 Features

* 🔍 Keyword-based research paper search
* 📚 Citation-aware recommendations
* 🖥️ Clean frontend interface for interaction
* ⚙️ Backend processing for query handling
* 📂 Modular structure (frontend + backend separation)

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Harsha-RVCEIAN/paper-recommender.git
cd paper-recommender
```

### 2. Setup backend

```bash
pip install -r requirements.txt
cd backend
python app.py
```

### 3. Run frontend

```bash
cd frontend
npm install
npm start
```
